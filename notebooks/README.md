# Notebooks

Ce dossier contient le notebook principal du projet BUPA.

## Contenu

- `BUPA_benchmark_final.ipynb` : notebook final du benchmark

## Rôle du notebook

Le notebook orchestre toute la pipeline :

- préparation
- données
- filtrage
- gold source
- génération de notes
- reconstruction
- scoring
- vérification
- analyse des erreurs
- export final
