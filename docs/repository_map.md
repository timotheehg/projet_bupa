# Carte du dépôt

## Objectif

Cette page explique où trouver chaque élément important du projet dans le dépôt GitHub.

## Racine du dépôt

### `README.md`
Page d’accueil du projet.

### `.gitignore`
Fichiers et dossiers exclus du suivi Git.

## `notebooks/`
Contient le notebook final du benchmark.

- [Notebook final](../notebooks/BUPA_benchmark_final.ipynb)

## `docs/`
Contient la documentation écrite directement pour GitHub.

- `project_overview.md`
- `pipeline_explained.md`
- `notebook_guide.md`
- `results_guide.md`
- `repository_map.md`
- `audit_summary.md`
- `benchmark_report.md`

## `results/`
Contient tous les résultats du benchmark.

### `results/intermediates/`
Résultats intermédiaires :
- source structurée
- notes
- reconstruction

### `results/summaries/`
Résultats finaux :
- métriques
- erreurs
- vérification

### `results/other_results/`
Résultats secondaires.

### `results/examples/`
Exemples patients.

## Parcours conseillé

Si tu veux comprendre le dépôt rapidement :

1. `README.md`
2. `docs/project_overview.md`
3. `docs/pipeline_explained.md`
4. `docs/results_guide.md`
5. `results/summaries/`
6. `results/examples/`
